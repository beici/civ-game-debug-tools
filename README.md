# civ-game-debug-tools

游戏轻量调试工具集，用于开发阶段快速查看、修改游戏数据、定位逻辑问题、提升开发与测试效率。

## 简介
civ-game-debug-tools 是面向文明策略类游戏的**轻量、可嵌入、无侵入**调试工具库，支持实时查看与修改玩家资源、城市数据、单位状态、地图信息、科技与政策进度等核心数据，方便开发者、测试与策划快速定位BUG、验证逻辑、调优数值。

## 主要功能
- 实时查看玩家、城市、单位、地块等游戏实体数据
- 支持直接修改资源、科技、文化、信仰、人口、产能等数值
- 快捷指令：立即完成建造、解锁科技、添加单位、切换回合等
- 游戏日志输出与过滤，方便定位逻辑异常与报错
- 轻量调试面板/控制台，可在游戏内直接调用
- 无侵入设计，不污染正式包逻辑，可快速开关

## 适用场景
- 游戏开发与逻辑调试
- 数值平衡与快速验证
- BUG复现与定位
- 策划快速测试功能流程
- 模组与二次开发调试

## 使用方式
集成到项目中，通过快捷键（如 F12）或自定义指令开启调试面板，输入指令或在界面上可视化操作，即可对游戏状态进行实时调试。
